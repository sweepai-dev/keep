## Basics

The Mock provider is a mock provider for testing purposes.

## Inputs

The Mock provider does not require any inputs.

## ProviderConfig

The Mock provider does not have any configuration options.

## Outputs

The Mock provider does not provide any outputs.

## BaseProvider

The Mock provider extends the BaseProvider. The BaseProvider provides basic functionality that all providers must have.

## Authentication Parameters

The Mock provider does not require any authentication parameters.

## Connecting with the Provider

To connect with the Mock provider, you simply need to instantiate it. You do not need to provide any inputs or authentication parameters.
